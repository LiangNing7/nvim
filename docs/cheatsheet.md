# Craftzdog Neovim 极客速查表

> 基于 **LazyVim** 框架，插件管理器为 **lazy.nvim**

---

## 一、基础架构

| 项目 | 内容 |
|------|------|
| **Leader 键** | `Space`（空格键） |
| **插件管理器** | `lazy.nvim`，启动时自动检查更新 |
| **配置框架** | `LazyVim`（预置大量合理默认值） |
| **自动补全引擎** | `blink.cmp` |
| **Picker 引擎** | `telescope.nvim`（`vim.g.lazyvim_picker = "telescope"`） |
| **内置警告** | 连按 `hjkl/+-` 超过 10 次会触发 "Hold it Cowboy!" 警告，强制你用更高效的移动方式 |

---

## 二、文件查找与检索（Telescope）

| 快捷键 | 功能 |
|--------|------|
| `;f` | 查找文件（含隐藏文件，遵守 .gitignore） |
| `;r` | 全局实时 Grep 搜索字符串（含隐藏文件） |
| `\\` | 列出所有打开的 Buffer |
| `;;` | 恢复上一次 Telescope 搜索状态 |
| `;e` | 列出所有诊断错误（Diagnostics） |
| `;s` | 通过 Treesitter 搜索函数名、变量等符号 |
| `;c` | 查看光标下符号的 LSP 入口调用链 |
| `;t` | 搜索 Vim 帮助文档标签 |
| `sf` | 以当前文件路径打开**文件浏览器**（telescope-file-browser） |
| `<leader>fP` | 在 lazy.nvim 插件目录中查找文件 |

**文件浏览器（`sf`）内部按键（Normal 模式）：**

| 按键 | 功能 |
|------|------|
| `N` | 新建文件 |
| `h` | 返回上级目录 |
| `/` | 进入搜索模式（Insert） |
| `<C-u>` / `<C-d>` | 快速上移/下移 10 项 |

## 三、文本编辑黑魔法

### 3.1 自动补全括号与标签

| 插件 | 来源 | 功能 |
|------|------|------|
| `nvim-autopairs` | LazyVim 默认内置 | 输入 `(` `[` `{` `"` `'` 自动补全对应闭合符 |
| `nvim-ts-autotag` | LazyVim lang/typescript 扩展 | 输入 HTML/JSX 标签自动闭合，重命名时同步修改 |

> 这两个插件开箱即用，无需额外操作。

### 3.2 Surround 操作（mini.surround，LazyVim 内置）

| 快捷键 | 功能 | 示例 |
|--------|------|------|
| `gsa` + 动作 + 符号 | **添加**包裹符号 | `gsaiw"` → 给单词加双引号 |
| `gsd` + 符号 | **删除**包裹符号 | `gsd"` → 删除光标处的双引号 |
| `gsr` + 旧符号 + 新符号 | **替换**包裹符号 | `gsr"'` → 把双引号换成单引号 |
| `gsf` | 查找下一个包裹符号 | — |
| `gsF` | 查找上一个包裹符号 | — |

### 3.3 寄存器安全操作（防止污染剪贴板）

| 快捷键 | 模式 | 功能 |
|--------|------|------|
| `x` | Normal | 删除单个字符，**不污染寄存器** |
| `<leader>d/D` | Normal/Visual | 删除，**不污染寄存器** |
| `<leader>c/C` | Normal/Visual | 修改，**不污染寄存器** |
| `<leader>p/P` | Normal/Visual | 从**上次复制**的内容粘贴（寄存器 `0`） |
| `dw` | Normal | 向前删除一个词（不污染寄存器） |
| `<C-a>` | Normal | 全选所有内容 |

### 3.4 增量/减量（dial.nvim 增强版）

| 快捷键 | 功能 |
|--------|------|
| `<C-a>` | 增加数字、日期、bool、semver、`let`↔`const` 等 |
| `<C-x>` | 减少（同上逻辑） |
| `+` | 等同 `<C-a>`（原生方式） |
| `-` | 等同 `<C-x>`（原生方式） |

## 四、LSP 与代码开发

### 4.1 核心跳转（LazyVim 默认 + 本配置覆盖）

| 快捷键 | 功能 |
|--------|------|
| `gd` | **跳转到定义**（通过 Telescope 展示，在新窗口打开） |
| `gr` | 查看所有引用（Go to References） |
| `gI` | 跳转到实现（Go to Implementation） |
| `gy` | 跳转到类型定义（Go to Type Definition） |
| `K` | **悬浮文档**（Hover Documentation） |
| `gK` | 函数签名帮助（Signature Help） |

### 4.2 代码行为与重构

| 快捷键 | 功能 |
|--------|------|
| `<leader>ca` | **代码行为**（Code Action） |
| `<leader>cr` | **重命名**（Rename，使用 `inc-rename.nvim` 实现实时预览） |
| `<leader>cf` | 格式化当前文件 |
| `:ToggleAutoformat` | 命令行切换自动格式化开关 |
| `<leader>i` | 切换 **Inlay Hints** 显示 |
| `<leader>r` | 将光标下的 HEX 颜色转换为 HSL 格式 |

### 4.3 诊断导航

| 快捷键 | 功能 |
|--------|------|
| `<C-j>` | 跳到**下一个**诊断错误 |
| `;e` | 在 Telescope 中列出所有诊断 |

### 4.4 Copilot AI 补全

| 快捷键 | 功能 |
|--------|------|
| `<C-l>` | **接受**完整建议 |
| `<M-l>` | 接受一个**单词** |
| `<M-S-l>` | 接受一**行** |
| `<M-]>` | 下一条建议 |
| `<M-[>` | 上一条建议 |
| `<C-]>` | 驳回建议 |

## 五、文件树与窗口管理

> 本配置**没有** nvim-tree，用 `telescope-file-browser`（`sf`）代替文件树，并 hijack 了原生 netrw。

### 5.1 分屏

| 快捷键 | 功能 |
|--------|------|
| `ss` | 水平分屏（`:split`） |
| `sv` | 垂直分屏（`:vsplit`） |

### 5.2 窗口跳转

| 快捷键 | 功能 |
|--------|------|
| `sh` | 跳到左侧窗口 |
| `sl` | 跳到右侧窗口 |
| `sk` | 跳到上方窗口 |
| `sj` | 跳到下方窗口 |

### 5.3 窗口大小调整

| 快捷键 | 功能 |
|--------|------|
| `<C-w><left>` | 向左缩小 |
| `<C-w><right>` | 向右扩大 |
| `<C-w><up>` | 向上扩大 |
| `<C-w><down>` | 向下缩小 |

### 5.4 专注模式

| 快捷键 | 功能 |
|--------|------|
| `<leader>z` | 切换 **Zen Mode**（全屏专注写代码） |

## 六、Tab 与 Buffer 管理

| 快捷键 | 功能 |
|--------|------|
| `te` | 新建 Tab（`:tabedit`，后接文件名） |
| `<Tab>` | 切换到**下一个** Tab（BufferLine） |
| `<S-Tab>` | 切换到**上一个** Tab（BufferLine） |
| `\\` | 用 Telescope 列出所有 Buffer 并跳转 |
| `<leader>th` | 关闭所有**隐藏**的 Buffer |
| `<leader>tu` | 关闭所有**无名**的 Buffer |

## 七、Git 操作

| 快捷键 | 功能 |
|--------|------|
| `<leader>gb` | 打开 **Git Blame** 视图 |
| `<leader>go` | 在浏览器中打开当前文件对应的 Git 仓库页面 |
